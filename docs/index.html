API Documentation

Endpoints

Category Method Endpoint Description
Authentication POST /auth/login Login and obtain access & refresh tokens
Authentication POST /auth/refresh Refresh access token
Authentication POST /auth/logout Logout and invalidate the refresh token
Admin - Lecturer POST /admin/lecturer Create a new lecturer
Admin - Student POST /admin/student Create a new student
Admin - Faculty POST /admin/faculty Create a new faculty
Admin - Major POST /admin/{faculty_id}/major Create a new major within a faculty
Admin - Room POST /admin/{department_id}/room Create a new room within a department
Admin - Course POST /admin/{major_id}/course Create a new course
Admin - Section POST /admin/{course_id}/section Create a new course section

POST /auth/login

This endpoint authenticates a user and returns JWT tokens.

Payload: 
{
  "identifier": "string",
  "password": "string"
}
Response: 
{
  "access_token": "string",
  "refresh_token": "string"
}

POST /auth/refresh

Payload: 
{
  "refresh_token": "string"
}
Response: 
{
  "access_token": "string",
  "refresh_token": "string"
}

POST /auth/logout

Authorization: Bearer {access_token}

Payload: 
{
  "refresh_token": "string"
}

Response: Empty body with HTTP status 204 (No Content).
